# Event Platform

A Event platform built at IgniteLab by Rocketseat, an Event for a front-end specialization using React.

Check here the <a href="https://ignite-lab-event-platform-five.vercel.app/" target="_blank">deployed app</a>

## Technology

* ReactJS
* TypeScript
* TailWind
* Vite
* GraphCMS
* GraphQL
